A fork from the archived repo of [demucs](https://github.com/facebookresearch/demucs)

This version replaces torchaudio with torchcodec.

**Installation:**

```
pip install -U demucs-torchcodec
```

**Usage**

```
demucs-torchcodec test.mp3
```
